# lrn-jb

A practice repository for learning and experimenting with **CSS frameworks**, **Sass**, and **SCSS**.

## Overview

This project is focused on improving frontend styling skills by exploring:

- CSS fundamentals
- Sass syntax (`.sass`)
- SCSS syntax (`.scss`)
- Variables, nesting, mixins, and functions
- Component styling structure
- Responsive layouts and UI experiments


## Learning Notes

This repository is mainly for:
- testing syntax
- comparing outputs
- experimenting with styling patterns

Not intended as a production application.
